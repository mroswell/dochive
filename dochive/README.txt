DocHive has 2 prerequisites, ImageMagick and Tesseract.

Install both of these, for command line use, and update 

(1) convert.bat
(2) draw.bat
(3) tesseract.bat

with a path like 

"%PROGRAMFILES%\ImageMagick-6.7.8-Q16\Convert" 
or
"%PROGRAMFILES%\Tesseract-OCR\tesseract"

Compile the java program by executing run.bat
